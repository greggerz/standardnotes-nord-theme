# Standard Notes Nord Theme

A Nord theme for Standard Notes. Based on the Nord Theme by arctic-studio.

![Preview](https://raw.githubusercontent.com/greggerz/standardnotes-nord-theme/main/preview.png)

## Installation

1. Open Standard Notes.
2. Go to the "Preferences" menu.
3. Selected "Plugins".
4. Scroll down to the bottom of the menu and paste the following URL
   into the "Install Custom Plugin" field:
   <https://listed.to/p/rJPRlkSjml>
